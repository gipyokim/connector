2.10. blobFiltering

BLOB 데이터를 필터링한 결과 문자열을 반환한다. 매개변수 o는 바이트 배열 타입이어야 한다. filename는 텍스트 필터로 확장자 정보를 줄 때만 사용된다.

String blobFiltering(Object o, String filename) throws Exception

매개변수

매개변수설명

o

BLOB 데이터

filename

파일명

반환값

  • 필터링 결과 문자열
