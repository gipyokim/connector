7. SharePoint 덤프

다음 옵션들은 Microsoft SharePoint 덤프 작업 시 사용하는 설정이다.

옵션설명

sps_source (필수)

SharePoint 데이터 소스

sps_source = {
    url = http://sps:41168
    domain = sharepoint.com
    user = foo
    password = bar
}

include_site

수집 대상 사이트

생략할 경우 전체 사이트를 대상으로 수집한다. 여러 사이트를 지정할 경우 콤마(,)로 구분한다.

include_site = QA01/NEW, QA01/QATEAM/QATEAM01

사이트 이름에 콤마(,)가 들어가는 경우는 역슬래시 콤마(\,)로 escape 처리해야 한다.

[사이트명]
사용자
게시판
그룹웨어,개발
그룹웨어,운영

[설정]
include_site = 사용자,게시판,그룹웨어\,개발,그룹웨어\,운영

exclude_site

수집 제외 사이트

여러 사이트를 지정할 경우 콤마(,)로 구분한다.

exclude_site = QA01/NEW308

사이트 이름에 콤마(,)가 들어가는 경우는 역슬래시 콤마(\,)로 escape 처리해야 한다.

[사이트명]
사용자
게시판
그룹웨어,개발
그룹웨어,운영

[설정]
exclude_site = 사용자,게시판,그룹웨어\,개발,그룹웨어\,운영

include_list

수집 대상 리스트

생략할 경우 전체 리스트를 대상으로 수집한다. 여러 리스트를 지정할 경우 콤마(,)로 구분한다.

include_list = 공유 문서

리스트 이름에 콤마(,)가 들어가는 경우는 역슬래시 콤마(\,)로 escape 처리해야 한다.

[리스트명]
사용자
게시판
그룹웨어,개발
그룹웨어,운영

[설정]
include_list = 사용자,게시판,그룹웨어\,개발,그룹웨어\,운영

exclude_list

수집 제외 리스트

여러 리스트를 지정할 경우 콤마(,)로 구분한다.

exclude_list = 팀사이트

리스트 이름에 콤마(,)가 들어가는 경우는 역슬래시 콤마(\,)로 escape 처리해야 한다.

[리스트명]
사용자
게시판
그룹웨어,개발
그룹웨어,운영

[설정]
exclude_list = 사용자,게시판,그룹웨어\,개발,그룹웨어\,운영

fetch_size

한 번에 패치할 레코드 개수 (기본값 : 128)

field_mapping (필수)

"필드 매핑"을 참고한다.

참고

  • 예외값 : 공백

temp_dir

첨부 파일을 내려받을 임시 디렉터리 경로 (기본값 : target_dir/temp)

이미 첨부 파일을 내려받았다면 변경 시각을 검사하여 다시 내려받을지 결정한다.

auth_type

첨부파일 다운로드시 인증 방법

  • basic : windows server IIS 설정에서 기본인증 사용시 설정

참고

미설정시 NTLM 인증 사용

auth_type = basic
