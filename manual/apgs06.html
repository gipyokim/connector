6. Groovy 스트립트로 여러 개의 칼럼 값을 변경해야 할 경우 어떻게 하나요?

여러 칼럼 값을 변경하기 위해서는 Map을 반환하도록 그루비 함수를 작성하면 됩니다. 아래는 file_names, file_count 두 칼럼의 값을 반환하는 예제입니다.

public Map getFileNamesAndCount(String path) {
    File directory = new File(path);
    String[] fnames = directory.list();
    Map result = new HashMap(2);
    result.put("file_names", StringUtils.join(fnames, "|");
    result.put("file_count", fnames.length);
    return result;
}
