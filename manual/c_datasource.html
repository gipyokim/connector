1. DataSource 생성

데이터베이스 정보를 조회하기 위해서는 다음과 같이 DataSource를 생성해야 한다.

  • 구문

    run.bat -dbqry connect -ds data_source_name -u username -p password -url jdbc_url -driver dirver
    구성요소설명
    data_source_nameJDBC 데이터소스 이름
    usernameJDBC 사용자 이름
    passwordJDBC 비밀번호
    jdbc_url

    JDBC URL

    입력방식은 다음 표를 참고한다.

    driver

    JDBC 드라이버

    참고

    자세한 사항은 지원하는 대상 데이터베이스는 어떤 것들이 있나요?를 참고한다.

  • 예제

    DataSource를 생성한다.

    run.bat -dbqry connect -ds oracleTmp -u kcon -p kcon -url jdbc:oracle:thin:@10.10.20.104:1521:KCON -driver oracle.jdbc.driver.OracleDriver

    다음과 같은 메시지가 출력된다.

    { "datasource" : "oracleTmp"}

    rc파일이 "{설치경로}/temp"에 생성된다.

    connector/temp/oracleTmp.tmp
{
  "driver" : "oracle.jdbc.driver.OracleDriver"
  "url" : "jdbc:oracle:thin:@10.10.20.104:1521:KCON"
  "user" : "kcon"
  "password" : "kcon"
}
