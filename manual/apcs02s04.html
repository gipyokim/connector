2.4. extractAstriskKey

좌절단 검색 시, LIKE 검색 대신 String List의 EQUAL 검색으로 대체할 수 있도록 ,로 구분되는 문자열을 반환한다. 최소 길이(s)부터 최대 길이(e)까지 좌절단 문자열 및 원본 문자열을 함께 반환한다.

String extractAstriskKey(String src, String s, String e)

예제

extractAstriskKey("좌절단검색", "1", "3")
---
반환값 :  "좌,좌절,좌절단,좌절단검색"

매개변수

매개변수설명

src

입력 문자열

s

최소 길이

e

최대 길이

반환값

  • 쉼표로 구분되는 좌절단 검색용 문자열
