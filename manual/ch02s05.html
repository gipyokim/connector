5. 디렉터리 구성

커넥터 설치 후 디렉터리 구성은 다음과 같다.

디렉터리명내용

bin

실행 스크립트, 라이선스 파일

bin/script

작업별 덤프 스크립트

conf

작업 설정 파일, 로그 설정 파일

conf/templates

덤프 스크립트 템플릿, 작업 정의 파일 템플릿

data

작업 결과 파일

lib

라이브러리

lib/extra

라이브러리 (옵션 파일)

logs

로그 파일

script

그루비 스크립트 파일

manual
