2.2. removeTag

HTML 문자열로부터 entity escape 변환 및 태그 제거한 문자열을 반환한다. SCRIPT 및 STYLE 태그에 대해서는 태그 내용도 함께 제거한다.

String removeTag(String htmlString)

예제

아래 문자열은 :

<html>
<head>
<script>document.getElementById("demo").innerHTML 
        = "Hello JavaScript!";</script>
<style>body {background-color: white;}</style>
</head>
<body>
<p id="demo">la langue &lt;Fran&ccedil;ais&gt;!</p>
</body>
</html>

다음과 같이 변경된다.

la langue <Français>!
