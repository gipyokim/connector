7. Groovy 스크립트의 로그를 별도 파일로 저장할 수 있나요?

Groovy 스크립트에서의 로그를 별도 파일로 저장하기 위해서는 아래와 같은 작업을 수행하면 됩니다.

  1. {설치 경로}/conf/log4j.properties 파일에서 그루비 로그 설정 관련된 주석을 풉니다.

    og4j.logger.com.konantech.connector.groovy=DEBUG, groovy
...
log4j.appender.groovy=org.apache.log4j.RollingFileAppender
log4j.appender.groovy.File=${app.home}/logs/groovy.log
log4j.appender.groovy.maxBackupIndex=10
log4j.appender.groovy.maxFileSize=10MB
log4j.appender.groovy.Append=true
log4j.appender.groovy.layout=org.apache.log4j.PatternLayout
log4j.appender.groovy.layout.ConversionPattern=%d{yyyy-MM-dd HH:mm:ss,SSS} %p %t %m%n

  2. 그루비 스크립트에 로거 및 로그 기록할 코드를 아래와 같은 방식으로 작성합니다.

    package com.konantech.connector.groovy;
..
public class Foo {
 private static final Logger log = LoggerFactory.getLogger(Foo.class);

 public String bar(String str) {
     log.debug("{}", str);
     ...
 }
}

  3. {설치경로}/logs/groovy.log 파일에 로그가 정상적으로 기록되었는지 확인합니다.

    2015-05-15 09:47:14,742 DEBUG pool-2-thread-1 Tweet Button
2015-05-15 09:47:14,742 DEBUG pool-2-thread-1 Twitter for iPhone
2015-05-15 09:47:14,742 DEBUG pool-2-thread-1 Samsung Mobile
