1. Notes 덤프 작업 시 비밀번호 자동 입력 설정을 할 수 없나요?

Notes 덤프 작업을 할 때마다 비밀번호를 입력해야 합니다.

D:\CNT_201\bin>run32.bat -test notes testnotes
[0E68:0002-11EC] The ID file being used is: C:\Users
\Administrator.QATEST\AppData\Local\Lotus\Notes\Data\admin.id
[0E68:0002-11EC] 비밀번호 입력(취소하려면 Esc 키를 누르십시오):

비밀번호 자동 입력 설정을 위해서 다음의 절차를 수행합니다.

  1. Notes Client가 설치된 디렉터리와 DATA 디렉터리를 사용자 변수 PAHT에 등록합니다.

    (예)
D:\IBM\Lotus\Notes
D:\IBM\Lotus\Notes\data

  2. {설치경로}/lib/extra/nextpwd.dll을 Notes Client가 설치된 경로에 위치시킵니다.

    (예) D:\IBM\Lotus\Notes\nextpwd.dll

  3. Notes Client 디렉터리 하위에서 *.id(계정 인증 파일)와 같은 경로에 password.txt를 생성합니다. password.txt에는 Notes Client 접속 비밀번호를 입력합니다.

    (예) D:\IBM\Lotus\Notes\data\password.txt

  4. Notes Client가 설치된 경로에 있는 notes.ini의 파일을 열어서 아래의 내용을 추가로 입력하여 등록합니다.

    EXTMGR_ADDINS = nextpwd
