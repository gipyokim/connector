6장. 덤프

이 장에서는 덤프 수행 및 덤프 파일을 확인하는 방법에 관해 설명한다.

지정된 덤프 작업을 시작한다.

run.bat [-start] [start-options] <start-args>
옵션설명

Start Options

  • -mi=<seconds> : 진행 상태를 기록할 주기를 설정한다. 설정하지 않을 경우 기본값은 1초

  • -ti =<minutes> : 덤프 타임아웃을 설정한다.

  • -s : 파일 시스템 스캔 작업만 수행한다. 필터링은 수행하지 않는다.

  • -d : 이전에 스캔된 파일 목록으로 필터링을 수행한다.

  • -v : 작업 진행 상태를 콘솔에 표시한다.

Start Arguments

  • type : 수행할 작업의 종류 (fsg | dbg | reproc | mongo | sps | notes | xml)

  • job : 수행할 작업 명

  • mode : 덤프 방식 (reproc 작업 시 생략)

    • bulk : 벌크 덤프

    • inc : 증분 덤프
