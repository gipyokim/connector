2.14. 그루비 함수 내에서 Datasource 설정을 참조하는 법

그루비 함수 내에서 JDBC 인터페이스 사용을 위해 데이터소스에 접근하는 방법을 지원한다. beforeProcess 메소드를 살펴보면 아래와 같이 데이터소스에 접근하는 코드가 있다.

import javax.sql.DataSource;
import com.konantech.connector.dbg.JdbcUtils;
...
DataSource ds = JdbcUtils.getDataSource("Oracle");

JdbcUtils.getDataSource는 데이터소스 이름으로 데이터소스에 검색하여 반환하는 메소드이다. 위의 예제와 같이 Oracle이란 이름의 데이터소스에 접근하기 위해서는 작업 설정 파일에 아래와 같이 Oracle이란 이름으로 데이터소스가 정의되어 있어야 한다.

data_source Oracle = {
    driver = oracle.jdbc.driver.OracleDriver
    url = jdbc:oracle:thin:@127.0.0.1:1521:KCON
    user = foo
    password = bar
}
