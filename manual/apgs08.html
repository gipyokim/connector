8. Groovy 스크립트를 이용하여 특정 레코드는 덤프하지 않도록 할 수 있나요?

Groovy 스크립트에서 조건만족시 SkipRecordException 을 발생시키면, 해당 레코드는 덤프에서 제외됩니다.

import com.konantech.connector.core.SkipRecordException;

public String setValue(String value){
  if("".equals(value)){
     throw new SkipRecordException("Skip Record Value: " + value);
  }
}

skip 된 레코드들은 log/skip.log에 pk값과 Exception 메세지를 함께 기록하게 됩니다. status파일에도 skip된 레코드 개수가 기록되고, 커넥터 종료 메세지에서 complete,error,skip 개수가 함께 출력됩니다.
