7. informix DB 사용 시 Cannot create PoolableConnectionFactory (Transactions not supported) 오류가 발생합니다.

해당 오류는 connection pool 사용 시 autocommit이 false로 되어있기 때문에 발생합니다.

(커넥터 기본설정이 false) informix DB가 No Logging Mode이면 connection 연결 시에 autocommit이 true 여야 connection이 가능합니다. 때문에 DB Mode 자체를 Logging Mode로 변경하거나, data_source 옵션에 poolable=false 옵션을 추가하거나, 혹은 data_source 옵션에 autocommit=true 추가로 설정하시길 바랍니다.
