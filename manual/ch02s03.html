3. 커넥터 서버 설치 조건

커넥터 서버 설치 조건은 아래와 같다.

기능 설명커넥터 (Connector)
탑재 모듈

  • Groovgy 2.0

지원 OS

Windows, Linux, Unix 등 JDK 1.7 자바 플랫폼을 지원하는 OS

권장 HW 사양CPU2 EA 이상
Core4 EA 이상 (CPU 당 2개 이상)
Memory1 GB 이상 (권장 2GB 이상)
HDD500 GB 이상

참고

  • CPU와 Memory는 본 시스템의 처리 스레드 개수와 관련 있다.

  • HDD는 덤프, 로그 등 본 시스템이 생성, 보관하는 파일 용량과 관련이 있다.

Microsoft SharePoint

Microsoft SharePoint의 경우 Windows 환경만 지원된다.

IBM Notes

검색엔진 서버 OS에 따라 아래와 같이 구성할 수 있으며, 이 외의 환경 구성 시에는 코난테크놀로지의 기술 지원을 통해 문의한다.

  • Windows 인 경우 (권장)

    Windows로 구성할 때 시스템이 가장 안정적이고 기술 지원이 원활하다.

    DominoSearch Engine [ Windows 환경 ]

    IBM Domino Server

    (Windows / Linux / Unix)

    검색엔진 + 커넥터 +

    IBM Notes Client

    참고

    IBM Notes Client는 32bit만 지원되므로 커넥터 역시 32bit로 구동되어야 한다.

  • Linux / Unix 인 경우

    Linux / Unix 환경으로 구성할 때는 IBM Domino Server가 필요하다.

    DominoSearch Engine [ Linux / Unix 환경 ]

    IBM Domino Server

    검색엔진 + 커넥터 +

    IBM Domino Server (Notes Client 역할)

    참고

    • 검색엔진에 구성되는 IBM Domino Server의 라이선스가 필요할 수도 있음을 고려하여 환경을 구성해야 한다.

    • IBM Notes Client 지원 OS는 아래와 같으며, 이 외의 OS에서 커넥터를 연동하기 위해서는 IBM Notes Client 역할을 하는 IBM Domino Server 구성이 필요하다.

      • Windows (연동 테스트됨)

      • Mac OS X

      • Red Hat Ent Linx RHEL Dsktp wWrkstn, SUSE Linux Enterprise Desktop (SLED), Ubuntu
