2.9. notesFiltering

주어진 파일 경로에 저장된 파일들을 필터링한 결과를 묶어 문자열로 반환한다. 필터링 이후 파일들은 삭제된다. filepaths 파라미터는 파일 경로를 작업 파일에 정의된 파일 구분자로 묶어 전달해야 한다. 필터링 결과는 작업 파일에 정의된 텍스트 구분자로 묶어 반환된다.

String notesFiltering(String filepaths) throws Exception

각 구분자는 작업 파일에 아래와 같이 정의한다.

separator = {
    file = |
    text = <##@@##>
}

매개변수

매개변수설명

filepath

필터링할 파일 경로를 파일 구분자로 묶은 문자열

반환값

  • 필터링 결과를 텍스트 구분자로 묶은 문자열
