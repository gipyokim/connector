2. 데이터베이스 정보 조회

테이블 목록을 조회할 수 있다.

  • 구문

    run.bat -dbqry -ds data_source_name query
    구성요소설명
    data_source_name

    생성한 JDBC 데이터 소스의 이름

    query

    데이터베이스 정보 조회를 위한 쿼리

    • 테이블 목록 조회

      SHOW TABLES

    • 테이블 스키마 조회

      EXPLAIN TABLE table_name

    • 테이블 데이터 조회

      SELECT 문

  • 예제

    • 테이블 목록 조회

      테이블 목록을 다음과 같이 조회한다.

      run.bat -dbqry -ds oracleTmp "show tables"

      다음과 같이 테이블 목록이 List형태로 출력된다.

      {"tables":["ds_code_t", "ds_company_t", "ds_person_t"]}

    • 테이블 스키마 조회

      테이블 스키마를 다음과 같이 조회한다.

      run.bat -dbqry -ds oracleTmp "explain table CNT_TABLE"

      다음과 같이 테이블 스키마가 JSON형태로 출력된다.

      {
        "columnCount":11,
        "fields": [
                {
                    "fieldName": "calendarId",
                    "type": "bigint(20) unsigned"
                },
                {
                    "fieldName": "name",
                    "type": "varchar(20)"
                },
                {
                    "fieldName": "description",
                    "type": "varchar(500)"
                }
            ]
}

    • 테이블 데이터 조회

      데이터를 다음과 같이 조회한다.

      run.bat -dbqry -ds oracleTmp "SELECT * FROM CNT_TABLE"

      다음과 같이 데이터를 조회할 수 있다.

      {
        "recordCount" : 2,
        "columnCount" : 2,
        "fields":[
            {
                "fieldName" : "VALUE",
                "fieldName2" : "VALUE"
            },{
                "fieldName" : "VALUE",
                "fieldName2" : "VALUE"
            }
          ]
}
