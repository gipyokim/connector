1. 실행 방법

커넥터 설치 경로 하위의 bin 디렉터리에서 실행 명령을 한다.

  • 실행 위치 : {설치경로}/bin

run.bat [-task] [task-options] [task-args]
옵션설정값

Task

  • start : 지정된 작업을 시작한다. (Task 생략시, 기본값=start)

  • test : 지정된 작업의 설정 파일을 검사한다.

  • conv : 기존 설정 파일로부터 작업 설정 파일 변환 작업을 수행한다.

  • encrypt : 입력된 문자열을 암호화한다.

  • decrypt : 입력된 문자열을 복호화한다.

Task Options

  • -version : 버전 정보를 출력한다.

  • -h, -? -help : 도움말을 출력한다. 특정 태스크에 대한 도움말을 표시하기 위해서는 -task -h와 같이 태스크 지정 후 -h 옵션을 추가한다.

  • -dL : Log Table을 이용하여 증분 덤프를 수행한다. 덤프 수행 후 -cL 옵션을 수행하여 Log Table 삭제를 수행하여야 한다.(2.1.11부터 지원)

  • -cL : 이전 증분 덤프 결과를 기준으로 Log Table 삭제를 수행한다.(2.1.11부터 지원)
