부록 F. File Type에 따른 필터링 방법

커넥터 버전별 파일 타입에 따른 필터링은 아래를 참조한다.

버전설명

JDBC Gateway

"file_type"을 HTML 또는 AUTO 로 작성한다.

(사용 예)
PROPERTY filter
{
 class_path = ../lib
 class_name = KTFFilterProperty
 option = "server" = "127.0.0.1:35698"
 option = "file_type" = "HTML"
 option = "action" = "B2B"
}

커넥터 1.x

두번째 파라미터가 파일 타입을 지정하는 부분이며, 지정하지 않으면 AUTO로 동작된다.

ioUtilsU.extractFiledata(byte[] data, String type);

  • AUTO로 필터링 수행

    (사용 예)
ioUtilsU.extractFiledata(temp.getBytes(), "");

  • HTML로 필터링 수행

    (사용 예)
ioUtilsU.extractFiledata(temp.getBytes(), "HTML");

커넥터 2.x

두번째 파라미터가 파일 타입을 지정하는 부분이며, 지정하지 않으면 AUTO로 동작된다.

FilterUtils.extractFiledata(byte[] data, String type);

  • AUTO로 필터링 수행

    (사용 예)
FilterUtils.extractFiledata(temp.getBytes(), "");
= FilterUtils.extractFiledata(temp.getBytes());
= FilterUtils.extractFiledata(temp.getBytes(), "AUTO");

  • HTML로 필터링 수행

    (사용 예)
FilterUtils.extractFiledata(temp.getBytes(), "HTML");

참고

MIME TYPE의 문서 필터는 AUTO로 설정하여 사용할 수 있다.
