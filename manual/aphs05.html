5. 덤프 파일의 특정 필드에 "[B@525e633b" 와 같은 값이 출력됩니다.

"[B@" 로 시작되는 값은 해당 칼럼의 값이 바이트 배열형이라는 것을 의미합니다. byte[]을 매개변수로 받는 그루비 함수를 작성하여 필드를 매핑합니다. 바이트 배열형으로 값을 전달하는 데이터 타입은 VARBINARY, BINARY, LONGVARBINARY, BLOB이 있고, MS-SQL의 경우 TIMESTAMP 등 여러 타입이 있으며 데이터베이스마다 다를 수 있습니다.

아래의 예제를 참고하십시오.

FIELDNAME = COLUMNNAME
=>
FIELDNAME = SampleGroovy.byteArrayToString(@COLUMNNAME@)
...
SampleGroovy.groovy
 
String byteArrayToString(byte[] ba) {
    return new BigInteger(ba).toString();
}
