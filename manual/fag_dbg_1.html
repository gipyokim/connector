1. 지원하는 대상 데이터베이스는 어떤 것들이 있나요?

아래 표에 있는 데이터베이스 외에도 JDBC 2.0.x 이상을 지원하는 데이터베이스들에 대해 덤프가 가능합니다.

DatabaseDriverURL

Oracle

oracle.jdbc.driver.OracleDriver

jdbc:oracle:thin:@host:port:SID

MySQL (Connector/J)

com.mysql.jdbc.Driver

jdbc:mysql://host:port/db-name

MariaDB

org.mariadb.jdbc.Driver

jdbc:mariadb://host:port/db-name

Microsoft SQL Server (Microsoft Driver)

com.microsoft.sqlserver.jdbc.SQLServerDriver

jdbc:sqlserver://host:port;database=db-name[;property=value[;...]]

Microsoft SQL Server (jTDS Driver)

net.sourceforge.jtds.jdbc.Driver

jdbc:jtds:sqlserver://host:port/db-name[;property=value[;...]] jdbc:jtds:sybase://host:port/db-name[;property=value[;...]]

DB2

com.ibm.db2.jcc.DB2Driver

jdbc:db2://host:port/db-name

Altibase

Altibase.jdbc.driver.AltibaseDriver

jdbc:Altibase://host:port/db-name

CUBRID

cubrid.jdbc.driver.CUBRIDDriver

jdbc:cubrid:host:port:db-name:[?property[&property]]

Tibero

com.tmax.tibero.jdbc.TbDriver

jdbc:tibero:thin:@host:port:db-name

Informix

com.informix.jdbc.IfxDriver

jdbc:informix-sqli://host:port/db-name:INFORMIXSERVER=server[;property=value[;...]]

PostgreSQL

org.postgresql.Driver

jdbc:postgresql://host:port/db-name[?property=value[&...]]

Sybase

com.sybase.jdbc4.jdbc.SybDriver

jdbc:sybase:Tds:host:port/db-name

참고

  • 커넥터에서 제공하는 sqljdbc는 4.0버전이며, sqljdbc 4.2버전으로 업그레이드 시 jre1.8 버전을 사용해야 한다. (jre1.8버전 이하에서는 예외사항이 발생할 수 있다.)

  • Altibase 5버전과 Altibase 5 이하의 버전을 함께 연동하려면 altibase5.jar 파일을 다운받아 사용해야 한다. 그 외에는 Connector에서 제공하는 altibase.jar 를 사용한다.
