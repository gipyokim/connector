부록 D. XPath 튜토리얼

XPath(XML Path Language)는 XML 문서에서 경로 상에 위치한 요소(elements)나 속성(attributes)을 배치하고 처리하는 방법을 기술하는 언어이다.

W3C 표준으로 XSLT와 XPointer에 사용되는 언어이다.

XPath는 XML 문서의 노드를 정의하기 위해 경로식을 사용하며, 수학 함수와 기타 확장 가능한 표현을 가지고 있다.
