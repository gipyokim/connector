7장. 암호화/복호화

설정 파일에 DB 접속 정보가 노출되어 있으므로 보안 문제에 민감한 경우 DB 접속 정보를 암호화할 필요가 있다.

data_source 설정의 url, user, password 옵션들은 암호화시켜 설정할 수 있다. 암호화하기 위해서는 아래와 같이 실행한다.

run.bat -encrypt mypassword
...
Encrypted text: Vt6AFio0H+NlNttny+TaTg==

유사한 방식으로 암호화된 문자열을 복호화할 수 있다.

run.bat -decrypt Vt6AFio0H+NlNttny+TaTg==
...
Decrypted text: mypassword

다음 암호화된 문자열을 데이터베이스 설정 파일(예. dbg.rc) 에 추가한다.

data_source = {
    driver = oracle.jdbc.driver.OracleDriver
    url = jdbc:oracle:thin:@127.0.0.1:1521:KCON
    user = foo
    password = ENC(Vt6AFio0H+NlNttny+TaTg==)
}

암호화된 문자열은 ENC()로 감싼다. 암호화된 문자열로 혹은 평문으로 존재할 수 있기 때문에 반드시 ENC()로 감싸야 한다.

참고

MongoDB의 경우도 같은 방법으로 사용할 수 있다.
