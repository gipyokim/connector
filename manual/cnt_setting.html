3장. 작업 설정

이 장에서는 커넥터 작업 설정에 관해 설명한다.

작업 설정 파일은 커넥터가 덤프 파일을 생성하는데 필요한 각종 정보를 설정한 파일이다.

작업 설정 파일은 conf 디렉터리에 "작업명.rc" 라는 이름으로 저장한다.

  • 설정 파일 저장 경로 : {설치경로}/conf/{name}.rc

  • 설정 파일 참고 템플릿 : {설치경로}/conf/templates
